# TechForcz Website

IT Consulting Services website for **techforcz.com**  
Hosted on: Netlify (Drag & Drop)

## Folder Structure

```
techforcz/
├── index.html       ← Main website file
├── sitemap.xml      ← Google sitemap
├── css/
│   └── style.css    ← All styles
├── js/
│   └── main.js      ← All JavaScript (chatbot, animations, form)
├── assets/          ← Add images/icons here in future
└── README.md        ← This file
```

## How to Deploy

1. Make your changes to any file
2. Go to netlify.com → your site
3. Drag and drop the entire `techforcz` folder
4. Done ✅

## Tech Used

- Pure HTML, CSS, JavaScript (no frameworks)
- Google Fonts: DM Serif Display, Syne, DM Sans
- Web3Forms for contact form
- Netlify for hosting
